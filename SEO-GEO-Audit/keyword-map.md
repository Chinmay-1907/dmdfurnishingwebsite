# Keyword Map — DMD Furnishing

**Date:** 2026-04-10
**Purpose:** Manual keyword→URL mapping for on-page targeting. No paid API data (DataForSEO, Ahrefs, SEMrush) was used to build this map — volume and difficulty are qualitative estimates based on industry knowledge of B2B commercial furniture search behavior.

**How to read this document:**
- Each cluster targets one **pillar page** (the best-ranking target for the primary keyword).
- Each cluster has **supporting pages** that should cross-link to the pillar and rank for long-tail variants.
- **Primary keyword** is the one-term commercial-intent phrase the pillar page should own.
- **Secondary keywords** are long-tail variants with intent that should appear naturally in H2s, body, and anchor text.
- **Semantic variants** are LSI / NLP coverage terms that should appear naturally in the body for topical relevance but aren't targets.
- **Current title / description** is a snapshot of the live state; **target title / description** is the recommended rewrite if the current one is off-topic.

Volume estimates use a coarse 5-bucket scale: **very low** (<100/mo US), **low** (100-500), **moderate** (500-2000), **high** (2000-10000), **very high** (>10000).

## Important constraint: do not keyword-stuff

Every proposed rewrite should pass these tests:
1. Primary keyword appears in title (1x) and description (1-2x), naturally
2. Description reads as a human sentence, not a string of keywords
3. No repeated words (e.g., "commercial furniture manufacturer for commercial furniture projects") — that's stuffing
4. Brand suffix preserved where the layout template doesn't already add it
5. Character counts: title ≤ 60 chars, description 140-160 chars
